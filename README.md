## Controls

Player 1:

* W and S to steer cannon
* E to shoot
* Q to control explosion time

Player 2:

* Up and down to steer cannon
* Space to shoot
* Alt / Option to control explosion time

Controllers:

* Left joystick or trigger buttons to steer cannon
* A to shoot
* X to control explosion time
